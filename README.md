# Termux VSCode
## Instaling
1. Set up git
```bash
pkg install git
```
2. Clone the git repository
```bash
git clone https://github.com/ywtsmd/TermuxVSCode.git
cd TermuxVSCode
```
3. Set up VSCode:
```bash
install_vscode.sh
```
4. Run VSCode:
```bash
chmod +x start_vscode.sh
./start_vscode.sh
```

Done!
